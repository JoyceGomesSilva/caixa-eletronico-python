## *Caixa Eletrônico em Python*

## 📌 Sobre o projeto

Este projeto é um **simulador de caixa eletrônico desenvolvido em Python**, criado com o objetivo de praticar conceitos fundamentais e intermediários da programação, aprendidos com o curso da Cisco Networking Academy.

O sistema simula operações bancárias básicas diretamente no terminal, permitindo que o usuário cadastre uma conta, realize login e execute transações de forma organizada.

---

## 🚀 Funcionalidades

- ✅ Cadastro de usuário  
- ✅ Login com validação de dados  
- ✅ Limite de tentativas de acesso  
- ✅ Depósito de valores  
- ✅ Saque com verificação de saldo  
- ✅ Visualização de saldo formatado (R$)  
- ✅ Histórico de transações  
- ✅ Interface organizada no terminal  
- ✅ Uso de orientação a objetos (POO)  

---

## 🧠 Conceitos aplicados

Durante o desenvolvimento, foram utilizados:

- Variáveis e tipos de dados  
- Estruturas condicionais (`if`, `elif`, `else`)  
- Laços de repetição (`while`)  
- Listas (para armazenar histórico)  
- Funções  
- Orientação a Objetos:
  - Classes (`Conta`, `Usuario`)
  - Métodos
  - Atributos  
- Validação de entrada  

---

## 🏗️ Estrutura do projeto

O sistema foi desenvolvido com duas classes principais:

### 👤 `Usuario`
Responsável por:
- Cadastro
- Login
- Controle de acesso

### 🏦 `Conta`
Responsável por:
- Saldo
- Depósitos
- Saques
- Histórico
